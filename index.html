GitHub Pages 自動更新
ページに新機能が追加されたらボタン1つで公開URLに反映
📌 使い方： このツールがアップデートされたら、下の「GitHubに反映」ボタンを押すだけ。1〜2分後に 公開URL に自動反映されます。

💡 一度入力すれば次回から自動入力されます（このPCのブラウザに保存）

🔑 トークンの取得方法
  1. GitHubにログイン → 右上アイコン → Settings
  2. 左メニュー最下部「Developer settings」
  3. Personal access tokens → Tokens (classic)
  4. 「Generate new token」→ repo にチェック → Generate
  5. 表示された ghp_... をコピーして上に貼り付け

⚠️ トークンは30日で期限切れ。切れたら同じ手順で再発行してください。
